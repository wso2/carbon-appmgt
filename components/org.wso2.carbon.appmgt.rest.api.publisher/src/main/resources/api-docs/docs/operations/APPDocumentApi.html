APPDocumentApi

Get App Documents

Get a list of documents belonging to an App.\n

URL

http://apis.wso2.com/api/appm/publisher/v1.1/apps/{appType}/id/{appId}/docs

HTTP Method

GET

Scope

appm:read

CURL Example

curl -H "Authorization: Bearer b0982cd2aacd463ff5f63cd5ebe58f4a" "http://127.0.0.1:9763/apps/webapp/id/8328392-d88d-dwu9-333r-r3"

Sample Response

HTTP/1.1 200 OK Content-Type: application/json { "previous": "", "list": [ { "visibility": "API_LEVEL", "sourceType": "INLINE", "sourceUrl": null, "otherTypeName": null, "documentId": "0bcb7f05-599d-4e1a-adce-5cb89bfe58d5", "summary": "This is a sample documentation for v1.0.0", "name": "PhoneVerification App Documentation", "type": "HOWTO" }, { "visibility": "API_LEVEL", "sourceType": "URL", "sourceUrl": "http://wiki.cdyne.com/index.php/Phone_Verification", "otherTypeName": null, "documentId": "4145df31-04f1-440c-8d08-68952874622c", "summary": "This is the URL for online documentation", "name": "Online Documentation", "type": "SAMPLES" } ], "next": "", "count": 2 }
