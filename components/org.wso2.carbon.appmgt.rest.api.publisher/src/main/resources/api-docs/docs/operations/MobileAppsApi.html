Retrieving mobile application binaries

Retrieving .apk, .ipa binaries for mobile apps.\n

URL

http://apis.wso2.com/api/appm/publisher/v1.0/apps/mobile/binaries/{fileName}

HTTP Method

GET

Scope

appm:create

CURL Example

curl -X GET -H "Authorization: Bearer 0763d28a85f20d6fd8954f955859055c" "http://localhost:9763/api/appm/publisher/v1.1/apps/mobile/binaries/O0lsxdlhR998Hhc.apk"

Sample Response

HTTP/1.1 200 OK
